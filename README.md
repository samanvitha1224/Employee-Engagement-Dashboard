# 📊 Employee Engagement & Burnout Analytics Dashboard

📌 Internship Project – Data Analytics  
📍 Role: Data Analyst Intern  

---

## 🧩 Business Problem

Organizations require structured workforce analytics to monitor engagement levels and detect burnout risk early.  
Traditional HR reporting lacks composite metric modeling and interactive visualization.

This project builds a measurable Engagement Index and rule-based Burnout Risk classification to enable actionable insights.

---

## 🧠 Methodology

1. Data Cleaning & Preprocessing using Pandas  
2. Engagement Index calculation using composite metric formula  
3. Burnout Risk classification using rule-based logic  
4. Dashboard development using Streamlit  
5. Visualization mapping and KPI design  

---

## ⭐ Project Highlights

- Composite Engagement Index Engineering  
- Burnout Risk Segmentation  
- Interactive Filters (Department & Overtime)  
- KPI Card Design

## 📖 Project Overview

This project analyzes employee engagement and burnout risk using structured data metrics and interactive dashboards.  
It enables workforce-level insights through composite index calculation and burnout classification.

---

## 🎯 Objectives

- Compute Employee Engagement Index  
- Classify Burnout Risk  
- Perform Department-wise analysis  
- Build Interactive Dashboard using Streamlit  

---

## 📊 Key Insights

- Engagement varies across departments  
- Overtime correlates with increased burnout risk  
- Moderate engagement levels show workforce stability  
- Interactive filtering supports HR decision-making
