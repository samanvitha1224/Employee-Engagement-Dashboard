# 📊 Employee Engagement & Burnout Analytics Dashboard

📌 Internship Project – Data Analytics  
📍 Role: Data Analyst Intern  

---

## 📖 Project Overview

This project analyzes employee engagement and burnout risk using structured data metrics and interactive dashboards.  
It enables workforce-level insights through composite index calculation and burnout classification.

---

## 🎯 Objectives

- Compute Employee Engagement Index  
- Classify Burnout Risk  
- Perform Department-wise analysis  
- Build Interactive Dashboard using Streamlit  

---

## 🛠 Tech Stack

- Python  
- Pandas  
- Streamlit  
- Power BI  
- VS Code  

---

## 🚀 How to Run

1. Clone repository:
```
git clone https://github.com/samanvitha1224/Employee-Engagement-Dashboard.git
```

2. Install dependencies:
```
pip install streamlit pandas matplotlib plotly
```

3. Run app:
```
streamlit run app.py
```
